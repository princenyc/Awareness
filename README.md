# Awareness Tracker

A simple local-first story cluster tracker for the Awareness project by Chartreuse Explosives.

## Purpose

Capture story clusters, main sources, and social conversation links in one place before building a full dashboard.

## Current Features

- Add story clusters
- Add source/social links to each story
- Group links by platform
- Mark links as featured
- Save data locally in browser storage
- Export data as JSON or Markdown

## Local Use

Open `index.html` in a browser.

## Warning

Data is saved in browser localStorage. Clearing browser data can delete saved clusters. Export JSON regularly.
